# Banking-sql-analysis


Intermediate-level SQL project on a retail banking dataset.

## Concepts Covered
- Multi-table JOINs
- Correlated Subqueries
- CTEs (single & chained)
- Window Functions (RANK, LAG, SUM OVER)
- Risk Classification with CASE

## Tools
PostgreSQL

## Queries (15 total)
Q1 - Total deposits per customer  
Q2 - High avg transaction customers  
Q3 - Transactions above branch average  
Q4 - Monthly transaction trend  
Q5 - Loan-to-Deposit ratio  
Q6 - Branch performance scorecard  
Q7 - Dormant accounts  
Q8 - Top 3 customers per branch  
Q9 - Risk tier classification  
Q10 - Running total of credits  
Q11 - Month-over-month growth  
Q12 - Customers with Savings + Loan  
Q13 - Duplicate transaction check  
Q14 - Customer balance segmentation  
Q15 - Full customer 360 view
